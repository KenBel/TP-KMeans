# TP – K-Means : Segmentation de Clients

Projet de segmentation de clients utilisant l’algorithme K-Means sur le dataset Mall Customers.
L’objectif est d’identifier différents profils de clients à partir de leur âge, revenu annuel et score de dépense.

Contenu du dépôt :
- kmeans.py : code principal du clustering K-Means
- dbscan.py : comparaison avec DBSCAN (bonus)
- Mall_Customers.csv : dataset (optionnel)
- Kmeans_Kenza_BELALLOUI.pdf : rapport complet du TP

Le projet inclut :
- préparation et analyse exploratoire des données
- choix du nombre optimal de clusters (méthode du coude + silhouette)
- visualisation 2D/3D des clusters
- interprétation des segments obtenus
